# 外部交互 {#title}

一些无法在应用内界面上直接使用的功能

## 跳转应用 {#scheme}

可以在浏览器或其他应用打开特定链接来跳转到应用内页面

| 页面     | 链接           |
| -------- | -------------- |
| 首页     | `gkd://page`   |
| 高级设置 | `gkd://page/1` |
| 快照记录 | `gkd://page/2` |

## 快捷开关 {#shortcut}

GKD 在通知栏面板声明了一些快捷开关, 可快捷使用一些功能

| 名称     | 功能                                         |
| -------- | -------------------------------------------- |
| GKD      | 打开或关闭无障碍权限                         |
| HTTP服务 | 打开或关闭HTTP服务                           |
| 悬浮按钮 | 打开或关闭悬浮按钮, 点击出现的按钮可捕获快照 |
| 捕获快照 | 点击后捕获当前应用界面快照                   |
